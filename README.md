# Birthday Bro

Static greeting site ready for GitHub Pages.

## How to publish

1. Create a new repository on GitHub.
2. Upload everything from this `birthday-bro` folder to the repository root:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `.nojekyll`
   - the `assets` folder
3. In GitHub, open `Settings` -> `Pages`.
4. Set `Source` to `Deploy from a branch`.
5. Choose branch `main` and folder `/root`.
6. Save. GitHub will show the public site link after the first deploy.

The music may require pressing the start button because browsers often block autoplay with sound.
